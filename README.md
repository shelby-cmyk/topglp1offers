# Top GLP-1 Offers

Forbes-style comparison listicle landing page for topglp1offers.com. Compares Thinnex, Brello Health, and DirectMeds with editor scores, price anchoring, a 60-second match quiz, 2-step lead form, and exit-intent capture.

## Local preview
Open `index.html` directly in a browser. No build step.

## Structure
- `index.html` — single-file page (HTML + inline CSS + JS)
- `assets/` — partner logos (Thinnex, Brello, DirectMeds)
